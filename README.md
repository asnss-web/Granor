# GRANOR — Modular Floating Docks

Готовый адаптивный одностраничный сайт для модульных плавучих доков GRANOR.

## Что внутри

- `index.html` — структура сайта
- `styles.css` — дизайн и адаптив
- `script.js` — меню, анимации и калькулятор
- `assets/` — исходные визуальные референсы
- `.nojekyll` — корректная публикация через GitHub Pages

## Как загрузить с iPhone

1. Скачайте ZIP из ChatGPT и откройте его в приложении **Файлы**.
2. Нажмите на ZIP — iPhone создаст распакованную папку.
3. Откройте репозиторий `Granor` в Safari на github.com.
4. Выберите **Add file → Upload files**.
5. Загрузите файлы **изнутри** папки, а не саму папку ZIP.
6. Нажмите **Commit changes**.

## Как включить сайт

1. В репозитории откройте **Settings → Pages**.
2. В `Build and deployment` выберите **Deploy from a branch**.
3. Branch: `main`.
4. Folder: `/ (root)`.
5. Нажмите **Save**.

После публикации адрес обычно будет:

`https://asnss-web.github.io/Granor/`

## Перед реальным запуском

Замените контакты и обработчик формы в `index.html`/`script.js`, добавьте реальные фотографии доков и подключите домен.
